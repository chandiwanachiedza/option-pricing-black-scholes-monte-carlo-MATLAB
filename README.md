Black–Scholes Option Pricing Model
Description
The Black–Scholes model calculates the theoretical price of European options.
The model assumes stock prices follow geometric Brownian motion.

Monte Carlo Black–Scholes Simulation
Description
Monte Carlo simulation estimates option prices by simulating many possible future stock prices.

Purpose
The Monte Carlo estimate should converge towards the analytical Black–Scholes value as the number of simulations increases.
The experiment compares:
Analytical Black–Scholes price
Monte Carlo estimated price
